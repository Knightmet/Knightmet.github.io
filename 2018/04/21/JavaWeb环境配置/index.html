首先你得有Java

在官网下载即可，安装即可（JDK,JRE都装一下）下载点这里

下载好以后配置环境变量

新建 JAVA_HOME 变量设置为jdk的安装路径
例 F:\JAVA\JavaJDK

找到系统变量path （win10双击点编辑文本) 在后面加上 ;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

然后安装JAVA开发环境 推荐Eclipse

根据系统是32位或64位，下载不同的版本。下载点这里

安装Tomcat

附上下载地址，推荐下载Core 下的最后一个安装版，一步到位，省的麻烦。

安装好以后，需要配置一下环境变量
新建 CATALINA_HOME 变量值设为Tomcat安装路径
例： C:\Program Files\Apache Software Foundation\Tomcat 7.0
新建 CATALINA_BASE 变量值设置和上面一样
找到系统变量path 在后面添加 ;%CATALINA_HOME%\lib;%CATALINA_HOME%\bin;

然后打开浏览器，在地址栏中输入http://localhost:8080 ，如果成功，就会出现那个经典的图片，，，就不附图了。

将Tomcat导入Eclipse中


新建一个Web项目

我这里新建的是 Dynamic Web Project (如果你建别的失败了，就跟着我来吧，走了好多弯路了)。



因为我安装的Eclipse可能和你的版本不一样，但具体的功能应该都有，所以可能图片和你的实际操作界面不一样，只要找对应的功能操作就可以了。

新建完成以后，在Java Resource 包下的 src 新建包或直接新建类 ,正常写Java程序即可。




注释

  • 如果你只是建了一个JAVA项目，貌似也可以写，需要引入Servlet.API. 还是贴一篇博客（这两个也是哦）

  • 对了，最好安装Java EE版本， 会省去不少麻烦。
